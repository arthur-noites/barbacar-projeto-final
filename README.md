1.1.1 O Projeto consiste na criação de um site dedicado a divulgação de uma empresa de revenda de automóveis. O projeto tem como objetivo promover a agência de veículos usados por meio de um site HTML moderno e atrativo, utilizando elementos em CSS, que transmita credibilidade e destaque os diferenciais da empresa, além disso, uma parte dedicada ao administrador em PHP. A plataforma apresentará informações completas sobre os automóveis disponíveis, incluindo descrições detalhadas, preços e imagens. Além disso, contará com canais de contato acessíveis, facilitando a comunicação entre clientes e a agência para agilizar o processo de atendimento e negociação. Do lado administrador será possível ainda adicionar, editar ou remover os veículos disponíveis, após passar por uma página de autenticação com login e senha.


Objetivo do projeto para uma empresa: 

Organizar, direcionar e ilustrar ao consumidor os produtos presentes em nosso catálogo para demonstrar nossa qualidade, valores e conceitos; 

Auxiliar no atendimento ao cliente para casos de duvidas sobre o produto que esteja em seu interesse;

Tecnologias Utilizadas:

• Frontend: HTML
• Estilização: CSS e React
• Banco de Dados e Autenticação: Xampp(através do MySQL)
• Ferramentas de Desenvolvimento: PHP
